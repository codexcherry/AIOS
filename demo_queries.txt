================================================================================
  QUICK REFERENCE — All Slash Commands
================================================================================

  /shell          AI-powered interactive shell
  /workflows      List all available workflows
  /run <name>     Execute a named workflow
  /ps             Live process table (CPU, RAM per process)
  /optimize       AI memory/CPU optimization suggestions
  /files <query>  Semantic file search
  /recent         Recently modified files
  /notes          List all saved notes
  /note <text>    Save a quick note
  /screenshot     Capture screen (auto-named)
  /screenshot <n> Capture screen with custom filename
  /time           Current time
  /date           Today's date
  /joke           Random joke
  /memory         Full context memory dump
  /context        Current session context
  /trash          List & restore soft-deleted files
  /kill <pid>     Terminate a process by PID
  /daemons        Show background daemon status
  /help           Full help screen
  /clear          Clear terminal
  /exit           Quit AIOS

================================================================================
  ARCHITECTURE SUMMARY (one-liner per layer)
================================================================================

  Config          →  pyproject.toml + ~/.aios/config.json, thread-safe reads
  Logger          →  Rich-formatted structured log, file + console
  Context Memory  →  JSON session store, file index, command log
  RL Memory       →  Correction learning, similarity scoring, prompt injection
  URL Router      →  20+ site regex rules, <1ms, zero LLM calls
  App Normalizer  →  200+ alias → canonical app name mapping
  App Launcher    →  Parallel subprocess launch via ThreadPoolExecutor
  File Ops        →  CRUD + AI content gen + soft-delete + path guard
  File Manager    →  3-stage: type-detect → parallel scan → LLM rank
  Process Manager →  psutil live stats + protected-process kill guard
  AI Shell        →  NL → shell cmd via LLM, dangerous pattern block
  Task Executor   →  Shared pool, sequential + parallel step groups
  Workflow Engine →  Preset + LLM-generated plans, step validation
  LLM Engine      →  Ollama HTTP wrapper, lru_cache, rl_context injection
  Features        →  Assistant / Notes / Screenshot (instant, no LLM)
  Daemons         →  4 background threads: monitor / organizer / guardian / watcher
